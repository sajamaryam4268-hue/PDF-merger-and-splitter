# PDF Merger & Splitter

## Description

PDF Merger & Splitter is a Python application that allows users to merge multiple PDF files into a single document and split a PDF into individual pages. This project demonstrates file handling and working with external Python libraries.

## Features

* Merge multiple PDF files
* Split PDF into separate pages
* Simple and easy to use
* Fast processing

## Technologies Used

* Python
* PyPDF2
